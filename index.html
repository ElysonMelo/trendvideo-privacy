Política de Privacidade - TrendVideo AI

Última atualização: 09 de julho de 2026

Esta Política de Privacidade descreve como o aplicativo TrendVideo AI ("nós", "o aplicativo") coleta, usa e protege as informações dos usuários.

1. Quem somos

O TrendVideo AI é uma ferramenta pessoal de automação de marketing, criada para identificar produtos em alta em marketplaces e gerar roteiros/vídeos de divulgação para publicação em redes sociais, incluindo o TikTok.

2. Quais dados coletamos

  • Dados de produtos em alta obtidos de APIs públicas de marketplaces (nome, preço, categoria).
  • Chaves de API fornecidas pelo próprio usuário para conectar-se a serviços de terceiros (ex: Mercado Livre, Shopee, TikTok).
  • Quando integrado ao TikTok: informações básicas de perfil (nome de usuário, avatar) e permissão para publicar conteúdo, conforme escopos autorizados explicitamente pelo usuário via OAuth.

3. Como usamos os dados

  • Para gerar roteiros de vídeo personalizados com base nos produtos selecionados.
  • Para publicar vídeos no TikTok em nome do usuário, apenas mediante autorização explícita (OAuth) e com prévia visualização do conteúdo pelo usuário antes da publicação.
  • Nenhum dado é vendido, compartilhado ou usado para fins de publicidade de terceiros.

4. Armazenamento e segurança

As chaves de API e credenciais são armazenadas localmente no ambiente do usuário (arquivo de configuração local), não sendo transmitidas a servidores externos além das próprias APIs oficiais às quais se destinam (Mercado Livre, Shopee, TikTok).

5. Compartilhamento de dados com o TikTok

Ao autorizar a integração com o TikTok, o aplicativo solicita apenas os escopos mínimos necessários para publicação de conteúdo (Content Posting API). O usuário pode revogar essa autorização a qualquer momento diretamente nas configurações da sua conta TikTok.

6. Direitos do usuário

  • Revogar o acesso do aplicativo a qualquer momento.
  • Solicitar a exclusão de qualquer dado armazenado localmente relacionado à sua conta.

7. Contato

Para dúvidas sobre esta política, entre em contato através do e-mail associado à conta de desenvolvedor deste aplicativo.
